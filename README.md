# ICAPS 2025 MiniZinc Tutorial: Constraint Modelling for Planning and Scheduling

This repository contains [MiniZinc](https://www.minizinc.org) examples and slides for the [2025 ICAPS Tutorial on MiniZinc:](https://icaps25.icaps-conference.org/program/tutorials/minizinc/) Constraint Modelling for Planning and Scheduling.

Authors:

- [Peter J. Stuckey](https://research.monash.edu/en/persons/peter-stuckey/)
- [Guido Tack](https://research.monash.edu/en/persons/guido-tack)

## Examples

The repository contains the MiniZinc code for the examples and exercises used this tutorial.

The following links will open the examples directly in the online [MiniZinc Playground](https://play.minizinc.dev/). Alternatively, you can clone or download this repository and open the example projects in the MiniZinc IDE.
